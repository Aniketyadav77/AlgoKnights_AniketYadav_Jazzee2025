# AlgoKnights_AniketYadav_Jazzee2025
🔊 JanSaarthi is a Gen-AI-powered multilingual assistant enabling rural citizens to access government schemes, file grievances, and get legal aid via mobile, WhatsApp, and toll-free IVR. Built using Bhashini, GPT-4, and CPGRAMS, it bridges the digital divide with voice-first, inclusive AI. Developed by Team AlgoKnights for Jazzee Hackathon 2025.
